# Site

## Backend

### Rodando localmente

Para executar o projeto de backend localmente deve-se executar os comandos abaixo:

```bash
cd backend
npm install
npm start
```

<br/>

Ao executar os comandos acima deve-se obter algo parecido com o log abaixo:

```
 npm start

> backend@1.0.0 start
> node index.js      

Server listening on 3001
```
